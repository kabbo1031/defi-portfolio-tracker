# Simple DeFi Portfolio Tracker

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- Show wallet balance
- Track multiple tokens
- Learn how DeFi dashboards work

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- MetaMask wallet
- Internet connection
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Click "Connect Wallet"
5. Click "Load Portfolio"

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Token settings  
- `wallet.js` = Wallet connect  
- `portfolio.js` = Portfolio reader  
- `ui.js` = Screen updates  
- `abi.js` = Token ABI  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs  

---

## Learning Goals

- Learn how DeFi apps read wallets
- Learn how token balances work
- Learn how blockchain data is shown in apps
- Learn how dashboards are built
